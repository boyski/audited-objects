Windows AO Client binary for Windows X86 (alpha)

These are the platforms for which the AO client is frequently built and tested. It should be buildable from source on other platforms too, but each platform tends to take a certain amount of porting work due to the low-level way in which the auditor works. Ports which have been done at various times include Mac OS X, FreeBSD, Solaris Sparc, and HP-UX. The Windows client is in proof-of-concept mode.
