Audited Objects Build Tool

This is the documentation for Audited Objects. For an introduction to what AO does, start with the Synopsis, then download the full install package or just a client binary, or get the source code.

To install AO for the first time, see the Installation instructions. If updating, see also the release notes. To install or update just a client, see the current set of AO Client Packages. Once server and client are installed and working you may want to look through the Tour Through Audited Objects and have a look at the FAQ.

Notes

  • Out of the box the AO server runs in evaluation mode, meaning that its data is retained only as long as the server is running. Turning on full persistence involves setting a few properties.
  • The Audited Objects tool set is still young; nothing here should yet be considered production quality. The Windows client is in an alpha state and has had only proof-of-concept testing. Also, no guarantee of compatibility is made between this and any subsequent releases, though there is no plan for incompatible changes.
